# Tier 3: LLM-as-Judge Evaluation

## Overview
Advanced assessment using Claude or other LLMs as judges:
- Prompt quality evaluation
- Agent reasoning assessment
- Explanation quality validation
- Scenario completeness check

## Commands
```bash
npm run test:tier3
```

## What's Evaluated
- Prompt effectiveness in teaching concepts
- Agent reasoning quality
- Explanation clarity and accuracy
- Scenario comprehensiveness
- Overall curriculum quality

## Implementation
Requires Claude API integration and custom evaluation prompts.
