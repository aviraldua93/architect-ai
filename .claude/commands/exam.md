# Mock Exam

Simulate a full Claude Certified Architect exam.

## Usage

```
/exam               — Full 30-question timed exam
/exam --time 45     — 45-minute time limit
```

## What It Does

1. Selects 30 questions weighted by domain importance
2. Starts a countdown timer
3. No feedback until submission — just like the real exam
4. Shows full score breakdown by domain at the end
5. Identifies weak areas to focus on

## Passing Score

720/1000 (approximately 72%)
